Cascading Style Sheets (CSS)

It is used to format the layout of a webpage.With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!



Note:

  • CSS stands for Cascading Style Sheets


  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media


  • CSS saves a lot of work. It can control the layout of multiple web pages all at once


  • External stylesheets are stored in CSS files



Using CSS


CSS can be added to HTML documents in 3 ways:



Note:

  • Inline - by using the style attribute inside HTML elements

    html


  • Internal - by using a "style" element in the "head" section

    head


  • External - by using a "stylesheet" element in the another css file
